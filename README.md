# react-demo-interview
这是一个简单的react demo，初学react 和 webpack。需要有nodejs环境。

－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－

主要用到：
  React with jsx;
  webpack ；
  less；
  
  
－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－
  
Install & run：

git clone https://github.com/greensoon/react-demo-interview
cd react-demo

－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－

npm install

－－－－－－－－－－－－－－－－－－－－－－－－－－－－－

打包：
webpack --progress --colors

－－－－－－－－－－－－－－－－－－－－－－－－－－－－－

本地服务启动：
npm start

－－－－－－－－－－－－－－－－－－－－－－－－－－－－－

测试：
在浏览器中输入 localhost:3000 进行测试。
